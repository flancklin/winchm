1、下载

直接用命令下载
rpm -Uvh http://dev.mysql.com/get/mysql-community-release-el7-5.noarch.rpm

下载好后查看有那些mysql文件，使用命令yum repolist enabled | grep "mysql.*-community.*"

2、安装

通过下面命令安装yum install mysql-community-server过程中直接点击Y就好直到出现
Complete!

3、启动

首先启动mysql systemctl start mysqld当然也可以用service mysqld start

4、配置

~】# mysql_secure_installation

  • Set root password?[Y/N]中文意思:是否要设置root用户的密码 选择Y【并配置root的密码】

  • Remove anonymous users? [Y/n]中文意思:是否删除匿名用户 选择N

  • Disallow root login remotely? [Y/n]中文意思:是否禁止root远程登录 选择N

  • Remove test database and access to it? [Y/n]中文意思:是否删除database数据库 选择N

  • Reload privilege tables now? [Y/n]中文意思:是否重新加载授权信息 选择Y


    • 5、远程ip授权

    当然想要远程还是要设置一下的
    GRANT ALL PRIVILEGES ON *.* TO 'root'@'%'IDENTIFIED BY 'password' WITH GRANT OPTION;
    第一个是你的用户名root，第二个password是远程的密码

    FLUSH PRIVILEGES
