‘urlManager’ 参数详解

'urlManager' => [
    'class' => 'yii\web\UrlManager',
    'enablePrettyUrl' => false,      //是否启用简写路由（就是不要参数r）
    'enableStrictParsing' => false,  //enablePrettyUrl=true才有效。是否严格按rules解析
    'showScriptName' => true,        //enablePrettyUrl=true才有效。
    'suffix'=>'',                    //enablePrettyUrl=true才有效。假后缀，比如 .html
    'routeParam'=>'r',               //enablePrettyUrl=false才有效。路由参数 index.php?r=site/index或index.php?r=moduleID/controllerID/actionID
    'normalizer'=>false,             //是否强制url标准化
    'ruleConfig' => ['class' => 'yii\web\UrlRule'],//解析rules的类
    'rules' => [
         'dashboard' => 'site/index',
         'POST <controller:[\w-]+>' => '<controller>/create',
         '<controller:[\w-]+>s' => '<controller>/index',
         'PUT <controller:[\w-]+>/<id:\d+>'    => '<controller>/update',
         'DELETE <controller:[\w-]+>/<id:\d+>' => '<controller>/delete',
         '<controller:[\w-]+>/<id:\d+>'   => '<controller>/view',
    ],
]
  • enablePrettyUrl : false 隐藏入口文件index.php, ex: /index.php/post/100 ==>>/post/100
  • enableStrictParsing : true 启动url严格匹配模式，匹配规则 yii\web\UrlManager::rules
  • rules : url 规则，注意定义不匹配时的默认路径（apache） yii\web\UrlRule
  • suffix : url 添加后缀名

‘rules’ 参数详解

1 url 正则匹配

  1. ### 示例1 url 正则匹配
  2.  
  3. 'rules' => [
  4.          posts/<year:\d{4}>/<category>' => 'post/index',
  5.         'posts' => 'post/index',
  6.         'post/<id:\d+>' => 'post/view',
  7.     ],

url示例1 解析

  • /index.php/posts : 匹配第二条规则,实际url post/index
  • /index.php/posts/2014/php:匹配第一条规则,实际urlpost/index参数year=2014，category=php
  • /index.php/post/100 : 匹配第三条规则,实际url post/view id=100
  • /index.php/posts/php : 不匹配任何规则,当’enableStrictParsing’ => false, 触发异常 yii\web\NotFoundHttpException!
     

2 url controller action正则匹配（常用）

  1. ### 示例2 url controller action正则匹配（常用）
  2.  
  3. 'rules' => [
  4.     '<controller:(post|comment)>/<id:\d+>/<action:(create|update|delete)>' => '<controller>/<action>',
  5.     '<controller:(post|comment)>/<id:\d+>' => '<controller>/view',
  6.     '<controller:(post|comment)>' => '<controller>/index',
  7.     ],

url示例2 解析

  • /index.php/comment/100/create:匹配第一条规则,实际url /comment/100/create 参数id=100
  • /index.php/comment/100 :匹配第二条规则,实际url /comment/view 参数id=100
  • /index.php/comments : 匹配第三条规则,实际url /comment/index

6 更高级路由配置

  1. ### 动态加载路由设置 bootstrap （module模块使用更多）
  2.  
  3. public function bootstrap($app)
  4. {
  5.     $app->getUrlManager()->addRules([
  6.         // rule declarations here
  7.     ], false);
  8. }
  9.  
  10.  
  11.  
  12. ###  自定义路由类
  13.  
  14. #配置
  15. 'rules' => [
  16.        '<controller:(post|comment)>' => '<controller>/index',
  17.         [
  18.             'class' => 'app\components\CarUrlRule', 
  19.         ],
  20.     ],
  21.  
  22.  
  23. #php实现
  24.  
  25. namespace app\components;
  26.  
  27. use yii\web\UrlRuleInterface;
  28. use yii\base\Object;
  29.  
  30. class CarUrlRule extends Object implements UrlRuleInterface
  31. {
  32.  
  33.     public function createUrl($manager, $route, $params)
  34.     {
  35.         if ($route === 'car/index') {
  36.             if (isset($params['manufacturer'], $params['model'])) {
  37.                 return $params['manufacturer'] . '/' . $params['model'];
  38.             } elseif (isset($params['manufacturer'])) {
  39.                 return $params['manufacturer'];
  40.             }
  41.         }
  42.         return false;  // this rule does not apply
  43.     }
  44.  
  45.     public function parseRequest($manager, $request)
  46.     {
  47.         $pathInfo = $request->getPathInfo();
  48.         if (preg_match('%^(\w+)(/(\w+))?$%', $pathInfo, $matches)) {
  49.             // check $matches[1] and $matches[3] to see
  50.             // if they match a manufacturer and a model in the database
  51.             // If so, set $params['manufacturer'] and/or $params['model']
  52.             // and return ['car/index', $params]
  53.         }
  54.         return false;  // this rule does not apply
  55.     }
  56. }
