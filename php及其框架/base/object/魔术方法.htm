__invoke()

          当尝试以调用函数的方式调用一个对象时，__invoke()方法会被自动调用。

 

class Test{
    public $a = 'a';
    public $b = 'b';

    function __construct()
    {
        var_dump(22);
    }
    function __invoke($one, $two)
    {
        var_dump([$one, $two]);
    }
}
$o = new Test();
$o(333,444);//333 444
