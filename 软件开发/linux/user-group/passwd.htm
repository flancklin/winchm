/etc/passwd中一行记录对应着一个用户，每行记录又被冒号(:)分隔为7个字段，其格式和具体含义如下：

用户名 ： 口令 ： 用户标识号 ： 组标识号 ： 注释性描述 ： 主目录 ： 登录时shell命令
root x 0 0 root /root /bin/bash
bin x 1 1 bin /bin /sbin/nologin

用户名：                 user账户。不超过8个字符。[a-zA-z0-9/]。不能有冒号：。为了兼容不建议有[.-+]等

口令：                      登录密码的加密串。/etc/passwd对全部用都【可读】。故这里用[x*]代替，真实的放在/etc/shadow中

用户标识号：          uid,用户ID。一个用户ID可以有多套【用户名-密码-主目录-登录shell命名】。取值范围0~65535。0-是root，1~99(1~499?)系统保留。

组标识号：              gid,群组ID。它对应/etc/group文件中的一条记录

注释行描述：          注释记录

主目录：                 登录成功后的其实工作目录
