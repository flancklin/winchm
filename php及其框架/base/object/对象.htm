一个类可以包含有属于自己的常量，变量（称为"属性"）以及函数（称为"方法"）。

 

final  如果父类中的方法被声明为 final，则子类无法覆盖该方法。如果一个类被声明为 final，则不能被继承。

 

self               当前自己。考虑上继承，不考虑下传承

$this/static    当前自己。指的是最终获得衣钵者(static调用的静态[属性、方法])

parent           指向当前（self）的父级,不考虑爷爷级祖先级，级太高请用全名称呼（类名）

 

在类定义内部，可以用 new self 、new static、  new parent 创建新对象

【若有构造函数new self(构造函数参数)，三个一样的】。

new self     构造一个完全一样的

new static   创建一个兄弟对象。

new parent 创建一个父类对象 




class Test
{
    public $a;

    function __construct($a)
    {
        $this->a = $a;
    }

    public function a(){
        return $this->a;
    }

    static public function cStatic($p)
    {
        return new static($p);
    }
    static public function cSelf($p){
        return new self($p);
    }
}

class Child extends Test
{
    static public function cParent($p){
        return new parent($p);
    }
}

$obj1 = new Test(1);
echo $obj1->a();           //1
$obj2 = new $obj1(2);
echo $obj2->a();           //2
var_dump($obj1 !== $obj2);

$obj3 = Test::cStatic(3);
echo $obj3->a();          //3

$obj4 = Test::cSelf(4);
echo $obj4->a();          //4

$obj5 = Child::cParent(5);
echo $obj5->a();          //5
