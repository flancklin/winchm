类的自动加载 类名与文件名【相同或有规则可循】

index.php与Test.php在同一目录下

 

index.php

<?php
spl_autoload_register(function ($class_name) {
    require_once './'. $class_name . '.php';
});

$o = new Test();
$o->index();
die;

Test.php

<?php
class Test{
    public function index(){
        echo "I am Test class";
    }
}

运行index.php。会输出    I am Test class.

但是在index.php中没有使用include或者require进行引入。

只是向spl_autoload_register写入了 【引入规则 】
