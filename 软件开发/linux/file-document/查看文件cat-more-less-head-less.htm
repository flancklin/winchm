打开一个文件。一次性读取全部内容在屏幕上显示
cat 帮助 cat --help
关闭/退出 先【Enter】键，再【Ctrl + D】
打开 cat [options]  文件名  -n 每行带编号
 -b 每行带编号(除了空白行)
 -s 把连续的多个空行显示为一行
 -E 把文件结尾放在可视区域，且每行结尾带上 $ 符号
 -u 忽略
 -v 显示非打印字符
 -T 显示TAB字符
【-A】 = 【-vET】
【-e】 = 【-vE】
【-t】 = 【-vT】
特殊 cat [option]  文件名1 文件名2 自动把多个文件拼接展示
查看 因为已经全部打印出来了。所以没有【翻页】【查找】【跳行】等操作了
如果必须查找。结合grep指令。【cat -n 文件名 | grep  查询关键词】
创建文件 【cat>文件名 】或【cat>>文件名】 >覆盖原文件。>>原文件结尾追加
【cat>文件名<<结束标志】按回车键
输入内容
【输入结束标志】回车		 创建文件并输入内容。结束标志自己定义，前后一致即可

 

 

 

 

一次性读取整个文件，但仅展示屏幕有限高度的数据。
less 帮助 less --help
带*表示可以跟数字参数。^是ctrl+的缩写。^e=ctrl+e
关闭/退出 q/:q/Q/:Q/ZZ
打开 less [options] [params] 文件名  -N 显示行号
 -s 把连续的多个空行显示为一行
 -z 设置窗口大小
 -I 搜索时候忽略大小写
 -i 若搜索词中没有大写，则忽略大小写
 -f 强制打开文件
翻看 * e/^e/j/^N/上箭头 向前一行(或n行)
* y/^y/k/^K/^P/下箭头 向后一行(或n行)
* f/^f/^V/空格/z/pageUp 前一窗口
* b/^B/ESC-v/w/pageDown 后一窗口
* d/^d 前半个窗口
* u/^u 后半个窗口
g 跳到文件首行
G 跳到文件最后
F 动态获取最后记录。如需退出ctrl+C
查找 * /模式 向前搜索
* ?模式 向后搜索
* n 重复上一次搜索
* N 反向重复上一次搜索
* ESC-n 重复上一次搜索,并跨越文件
* ESC-N 反向重复上一次搜索，并跨越文件
ESC-u 取消搜索的高亮显示

 

 

 

 

 

仅展示最后几行的内容。默认10行
tail 打开 tail [options] [params] 文件名  -c [+]NUM 输出后NUM个字符。(+表示除了最前面NUM个字符，其他全输出）
 -n [+]NUM 输出后NUM行。(+表示除了最前面NUM-1行，其他全输出）
 -q 不显示文件名(和默认一样)
 -v 第一行显示文件名
 -f 动态获取最后几条记录(默认每秒更新一次)
 -s 与-f搭配使用.设置间隔跟新的秒数
 --paid NUM 与-f搭配使用。追踪文件更新，直到进程NUM结束
head [options] [params] 文件名1 文件名2 同时打开多个文件，同时展示

 

 

 

 

一页一页的展示文件内容
more 帮助 more --help
关闭/退出 q
打开 more [options] 文件名  +NUM 从第n行开始显示
 -NUM 每屏显示n行
 +/string 搜索string并从该处前两行开始展示
 -s 把连续的多个空行显示为一行
 -c/-p 每页清屏展示，而不是滚屏(上一页接下一页)
 -d 显示提示【空格下翻。q退出】。仅提示作用
 -u 不显示下划线(如果有下划线的话)
翻看 b 上一窗口
空格 下一窗口(文档结束自动结束more指令)
Enter 下一行
无 上一行
 = 显示当前行号
：f 显示当前文件以及当前行号

 

 

 

 

 

 

 

仅展示前面几行的内容。默认10行
head 打开 head [options] [params] 文件名  -c [-]NUM 输出前NUM个字符。(-表示除了最后NUM个字符，其他全输出）
 -n [-]NUM 输出前NUM行。(-表示除了最后NUM行，其他全输出）
 -q 不显示文件名(和默认一样)
 -v 第一行显示文件名
head [options] [params] 文件名1 文件名2 同时打开多个文件，同时展示
